1. Purpose

Version is a task which automatically generates a compiled class encapsulating version and build number information. Version information are given as attribute values in the build files. The build number is automatically updated on every build.

2. Usage

As usual, a taskdef declaration must be put in the build file: 

    <taskdef name="version" classname="org.sadun.util.ant.Version" classpath="lib\version.jar"/>
